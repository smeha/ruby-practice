# Ruby language practice project
The project is to practice Ruby language

## Tech Stack
- Ruby (v3.4)
- RSpec (v3.13)
- RuboCop (v1.85)

## How to run locally
### Install dependencies
```bash
bundle install
```

### Run the project
```bash
ruby main.rb
```
- Chose from the list of available exercises

## Run lint tests
```bash
rubocop
```

### Auto-fix safe offenses
```bash
rubocop -a
```

## Run test cases
```bash
rspec
```
